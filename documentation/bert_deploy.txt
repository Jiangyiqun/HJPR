1 Clone the repo.
2 Make sure that python3 is installed in your system.
3 Go to the base of HJPR/
4 On the terminal, run the command [pip3 install -r requirements.txt]
4.5 To test whether all the packages required for BEFT model is installed, go to the base of HJPR/backend, and run the command [python testBert.py]. If success, it will return a list of course and printout text "Test success!" to terminal.
5 To run the the backend, go to the base of HJPR/backend, and run the command [python main.py]
